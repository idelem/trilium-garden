zim wiki compared with trilium:

  • (+) flat file system
  • (+) better export to html
    • has backlinks
    • custom template, allowing for custom navigation
  • (+) (somewhat) better shortcuts
  • (+) makes new pages on the fly, with multiple naming conventions
  • (+) manages todo lists like roam

 

  • (-) no clone（用了其它笔记才知道 trilium 的 clone 有多香）
  • (-) heeeeeeavvvvy configuration
  • (-) weird interface
  • (-) no native link map
  • (-) graphviz is unintuitive; doesn't render utf-8
  • (-) trilium has better shortcuts for creating new subnotes and siblings

zim 特色功能：

  • Ctrl Shift L 复制到当前页面的链接，Ctrl V 在其他页面粘贴
  • Ctrl D 插入日期，可以选择链接到 journal
  • Alt ← → 切换历史记录，↑↓ 切换目录里上下级笔记（感觉很方便小说写作）
  • Ctrl P 在浏览器里预览
